# Projeto 3 — CCF396 Jogos e Gamificação

**Descrição:** Este repositório contém o material e artefatos do Projeto 3 da disciplina **CCF396 - Jogos e Gamificação** (2025). O projeto consiste na criação e implementação de melhorias do clássico *Kaboom!* (Atari 2600) utilizando a engine **Godot 4**, adicionando novas mecânicas e funcionalidades.
